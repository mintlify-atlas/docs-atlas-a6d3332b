## Overview

The `tauri-plugin-audio-recorder` is a custom-built Tauri plugin that provides cross-platform audio recording capabilities using:
- **cpal**: Cross-platform audio library for device enumeration and stream capture
- **hound**: WAV file writer for lossless audio output
- **Multi-threading**: Dedicated recording thread to avoid blocking the main application

## Desktop Implementation (desktop.rs)

### Multi-Threading Architecture

<Info>
  cpal's `Stream` and `WavWriter` are **not Send**, so they cannot be shared across threads. The plugin uses a dedicated recording thread that owns these types, communicating via message passing.
</Info>

#### Shared State

```rust
struct SharedState {
    is_recording: AtomicBool,
    is_paused: AtomicBool,
    duration_ms: AtomicU64,
    output_path: Mutex<Option<String>>,
    sample_rate: AtomicU64,
    channels: AtomicU64,
}
```

This state is wrapped in `Arc` and shared between the main thread and recording thread.

### Initialization

```rust
pub fn init<R: Runtime, C: DeserializeOwned>(
    app: &AppHandle<R>,
    _api: PluginApi<R, C>,
) -> crate::Result<AudioRecorder<R>> {
    let shared = Arc::new(SharedState::default());
    let (cmd_tx, cmd_rx) = mpsc::channel::<RecorderCommand>();
    
    // Spawn the recording thread
    let shared_clone = Arc::clone(&shared);
    let handle = thread::spawn(move || {
        recording_thread(cmd_rx, shared_clone);
    });
    
    Ok(AudioRecorder {
        app: app.clone(),
        shared,
        cmd_tx,
        _thread_handle: Mutex::new(Some(handle)),
    })
}
```

### Recording Thread

```rust
fn recording_thread(rx: mpsc::Receiver<RecorderCommand>, shared: Arc<SharedState>) {
    let mut current_stream: Option<cpal::Stream> = None;
    let mut current_writer: Option<Arc<Mutex<Option<WavWriter<BufWriter<File>>>>>> = None;
    let mut write_flag: Option<Arc<AtomicBool>> = None;
    
    loop {
        match rx.recv() {
            Ok(RecorderCommand::Start(config, reply)) => {
                let result = start_recording_internal(&config, &shared);
                match result {
                    Ok((stream, writer, flag)) => {
                        current_stream = Some(stream);
                        current_writer = Some(writer);
                        write_flag = Some(flag);
                        let _ = reply.send(Ok(()));
                    }
                    Err(e) => {
                        let _ = reply.send(Err(e));
                    }
                }
            }
            Ok(RecorderCommand::Stop(reply)) => {
                let result = stop_recording_internal(
                    &shared,
                    &mut current_stream,
                    &mut current_writer
                );
                write_flag = None;
                let _ = reply.send(result);
            }
            Ok(RecorderCommand::Pause(reply)) => {
                if let Some(ref flag) = write_flag {
                    flag.store(false, Ordering::SeqCst);
                    shared.is_paused.store(true, Ordering::SeqCst);
                    let _ = reply.send(Ok(()));
                } else {
                    let _ = reply.send(Err(Error::NotRecording));
                }
            }
            Ok(RecorderCommand::Resume(reply)) => {
                if let Some(ref flag) = write_flag {
                    flag.store(true, Ordering::SeqCst);
                    shared.is_paused.store(false, Ordering::SeqCst);
                    let _ = reply.send(Ok(()));
                } else {
                    let _ = reply.send(Err(Error::NotRecording));
                }
            }
            Ok(RecorderCommand::Shutdown) | Err(_) => {
                // Clean up and exit
                if let Some(stream) = current_stream.take() {
                    drop(stream);
                }
                if let Some(writer) = current_writer.take() {
                    if let Ok(mut guard) = writer.lock() {
                        if let Some(w) = guard.take() {
                            let _ = w.finalize();
                        }
                    }
                }
                break;
            }
        }
    }
}
```

### Pause/Resume

Pause and resume are implemented via an `AtomicBool` flag that the stream callback checks:

```rust
// Pause
if let Some(ref flag) = write_flag {
    flag.store(false, Ordering::SeqCst);
    shared.is_paused.store(true, Ordering::SeqCst);
}

// Resume
if let Some(ref flag) = write_flag {
    flag.store(true, Ordering::SeqCst);
    shared.is_paused.store(false, Ordering::SeqCst);
}
```

The stream continues capturing audio, but samples are discarded when `write_flag` is `false`.

### Device Enumeration

```rust
pub fn get_devices(&self) -> crate::Result<AudioDevicesResponse> {
    let host = cpal::default_host();
    let default_device = host.default_input_device();
    let default_name = default_device.as_ref().and_then(|d| d.name().ok());
    
    let devices = host.input_devices()
        .map_err(|e| Error::Recording(format!("Failed to enumerate devices: {}", e)))?;
    
    let mut result = Vec::new();
    for device in devices {
        if let Ok(name) = device.name() {
            let is_default = default_name.as_ref().map(|n| n == &name).unwrap_or(false);
            result.push(AudioDevice {
                id: name.clone(),
                name,
                is_default,
            });
        }
    }
    
    Ok(AudioDevicesResponse { devices: result })
}
```

### Example: Recording Workflow

```typescript
import { invoke } from '@tauri-apps/api/core'
import { appDataDir, join } from '@tauri-apps/api/path'

class AudioRecorder {
  private isRecording = false
  
  async start() {
    if (this.isRecording) return
    
    const timestamp = Date.now()
    const appData = await appDataDir()
    const outputPath = await join(appData, 'recordings', `recording-${timestamp}`)
    
    try {
      await invoke('plugin:audio-recorder|start_recording', {
        config: {
          output_path: outputPath,
          quality: 'Medium'  // 44.1kHz, mono
        }
      })
      this.isRecording = true
      console.log('Recording started')
    } catch (error) {
      console.error('Failed to start recording:', error)
    }
  }
  
  async stop(): Promise<string | null> {
    if (!this.isRecording) return null
    
    try {
      const result = await invoke<RecordingResult>('plugin:audio-recorder|stop_recording')
      this.isRecording = false
      console.log(`Recorded ${result.duration_ms}ms to ${result.file_path}`)
      return result.file_path
    } catch (error) {
      console.error('Failed to stop recording:', error)
      return null
    }
  }
  
  async pause() {
    if (!this.isRecording) return
    await invoke('plugin:audio-recorder|pause_recording')
  }
  
  async resume() {
    if (!this.isRecording) return
    await invoke('plugin:audio-recorder|resume_recording')
  }
  
  async getStatus() {
    return await invoke<RecordingStatus>('plugin:audio-recorder|get_status')
  }
}

// Usage
const recorder = new AudioRecorder()
await recorder.start()
// ... record audio ...
const filePath = await recorder.stop()
```

## Troubleshooting

### No Audio Devices Found

```rust
Error::DeviceNotFound
```

**Solution:** Ensure a microphone is connected and enabled in system preferences.

### macOS Mutes Other Apps

<Warning>
  If Discord/Zoom/etc. are muted when recording, the plugin is forcing a sample rate instead of inheriting the hardware rate.
</Warning>

**Fix:** Ensure you're using the latest version with the hardware rate inheritance fix (`desktop.rs:156-171`).

### Permission Denied (macOS)

**First run:** The OS will prompt for microphone access. Grant it.

**Subsequent runs:** If denied, go to **System Preferences → Security & Privacy → Microphone** and enable Creative!.

### WAV File Corrupted

If the app crashes during recording, the WAV file may be incomplete (missing header).

**Prevention:** Always call `stop_recording()` before exiting. The plugin handles cleanup in the `Drop` implementation:

```rust
impl<R: Runtime> Drop for AudioRecorder<R> {
    fn drop(&mut self) {
        let _ = self.cmd_tx.send(RecorderCommand::Shutdown);
        if let Ok(mut guard) = self._thread_handle.lock() {
            if let Some(handle) = guard.take() {
                let _ = handle.join();
            }
        }
    }
}
```

**Reference:** `src-tauri/tauri-plugin-audio-recorder/src/desktop.rs:505-516`
